# Registro de identificadores persistentes

Este arquivo será atualizado após a publicação no Zenodo.

| Identificador | Valor | Uso |
|---|---|---|
| DOI específico da versão 1.0 | Aguardando publicação | Citação da edição imutável v1.0 |
| DOI conceitual da série | Aguardando publicação | Citação geral da série e versões futuras |
| GitHub Release | `axion-relatorios-v1.0` | Distribuição e rastreabilidade do código-fonte documental |

Não inserir um DOI provisório ou inferido. Os valores deverão ser copiados diretamente da página do registro publicado no Zenodo.
