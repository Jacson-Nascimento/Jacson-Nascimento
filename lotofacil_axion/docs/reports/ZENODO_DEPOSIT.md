## Fluxo de preservação

1. Conectar a conta GitHub em **Zenodo > Profile > Linked accounts**.
2. Abrir **Zenodo > GitHub**, selecionar **Sync now** e habilitar o repositório `Jacson-Nascimento/Jacson-Nascimento`.
3. No GitHub, publicar o rascunho da Release `axion-relatorios-v1.0`.
4. Aguardar o processamento da Release pelo Zenodo.
5. Conferir título, autoria, descrição, licença, palavras-chave e arquivos.
6. Registrar no repositório o DOI específico da versão 1.0 e o DOI conceitual da série.

## DOI e versionamento

- O **DOI específico** identifica os arquivos imutáveis da versão 1.0.
- O **DOI conceitual** deve ser usado para citar a série de forma geral e apontará para a versão mais recente.
- Versões futuras devem ser publicadas como novas GitHub Releases e novos registros vinculados no Zenodo.
